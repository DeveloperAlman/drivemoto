### Only Clean Technologies Used

- **HTML**: The project utilizes clean HTML5 (BEM Methodology) for the structure of web pages, ensuring a semantic and accessible foundation.

- **SCSS (Sass)**: Styling is implemented using SCSS, a popular preprocessor scripting language for CSS. This allows for the use of variables, nesting, and other features to streamline the styling process.

- **JavaScript**: For dynamic and interactive features, JavaScript has been employed. This ensures a smooth user experience and enables functionality such as product filtering and interactive forms.

### Responsive Design

The Drivemoto webshop is designed to be responsive, ensuring a seamless experience across a variety of devices and screen sizes. Media queries and flexible layouts have been implemented to adapt the content for optimal viewing on desktops, tablets, and mobile devices.

### Browser Compatibility

Great care has been taken to ensure compatibility with modern web browsers. The webshop is expected to perform well on popular browsers, including Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.

### Code Quality

The codebase adheres to best practices and coding standards. This ensures maintainability and makes it easier for developers to collaborate on the project.
